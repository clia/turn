# clia-turn

This is a temporary personal [webrtc-rs turn](https://github.com/webrtc-rs/webrtc/tree/master/turn) mod, support async AuthHandler.

It has no relation to [webrtc-rs](https://github.com/webrtc-rs/) authors.
